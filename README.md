# Map Bot

Telegram бот для отображения городов на карте.

## Описание

Бот позволяет пользователям искать города на карте мира, сохранять их в избранное и просматривать все сохраненные города на одной карте. Проект использует библиотеки Cartopy и Matplotlib для визуализации географических данных.

## Установка

1. Клонируйте репозиторий:
```bash
git clone <url-репозитория>
cd map-bot
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install pyTelegramBotAPI matplotlib cartopy
```

5. Настройте конфигурацию:
   - Откройте файл `config.py`
   - Добавьте токен вашего бота:
   ```python
   TOKEN = 'ваш_токен_бота'
   DATABASE = 'database.db'
   ```

6. Создайте базу данных:
```bash
python logic.py
```

7. Запустите бота:
```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/help` | Показать список команд |
| `/show_city <название>` | Показать город на карте |
| `/remember_city <название>` | Сохранить город в избранное |
| `/show_my_cities` | Показать все сохраненные города |

## Примеры использования

**Показать город:**
```
/show_city Moscow
```
Бот отправит карту с отмеченной Москвой.

**Сохранить город:**
```
/remember_city London
```
Бот добавит Лондон в список избранных.

**Показать избранные города:**
```
/show_my_cities
```
Бот отправит карту со всеми сохраненными городами.

## База данных

Проект использует SQLite с двумя таблицами:

- **cities** - содержит список городов с координатами
  - `id` - уникальный идентификатор
  - `city` - название города
  - `lat` - широта
  - `lng` - долгота

- **users_cities** - связывает пользователей с их избранными городами
  - `user_id` - идентификатор пользователя Telegram
  - `city_id` - ссылка на город из таблицы cities

## Технологии

- **pyTeleBotAPI** - взаимодействие с Telegram API
- **SQLite3** - хранение данных пользователей и городов
- **Matplotlib** - создание карт и визуализация
- **Cartopy** - географические проекции и картография

## Требования

- Python 3.7 или выше
- Библиотеки: pyTeleBotAPI, matplotlib, cartopy
